# AzureGraph

Azure Graph example projects



## First example Graph API subscriptions & SCIM client

The Project consists of 2 parts

* PowerShell script to create subscription for user changes
* Web API Project for Azure Web App to receive notifications and SCIM requests and write them to storage account
* https://learn.microsoft.com/en-us/graph/api/resources/change-notifications-api-overview?view=graph-rest-1.0
* https://learn.microsoft.com/en-us/azure/event-grid/subscribe-to-graph-api-events?tabs=http
